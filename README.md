#  Carrito de Compras

Proyecto final del curso de JavaScript.  
Consiste en un simulador de compra online donde el usuario puede ver productos, agregarlos al carrito y finalizar una compra simulada.

## Funcionalidades
- Productos cargados dinámicamente desde **data.json** usando `fetch()`.
- Renderizado con **DOM** y manejo de eventos.
- Carrito de compras con **localStorage**.
- Mensajes interactivos con **SweetAlert2**.
- Página de menú y página de carrito.
